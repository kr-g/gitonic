# gitonic 

gitonic simplifies the daily work with multiple git repositories.

gitonic comes with an easy to use Tkinter GUI.

# file status staged / ustaged 

the file status is the same as when calling 
[`git status --porcelain`](https://git-scm.com/docs/git-status#_output)


| status | comment |
|---|---|
| M | modified |
| A | added |
| D | deleted |
| R | renamed |
| C | copied |
| U | updated but unmerged |
| ?? | not in git |
